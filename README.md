# Sudoku

Projeto Sudoku em C.

## Estrutura

- A raiz do repositório contém este README.
- O código-fonte está em `sudoku/`.

## Como compilar

No terminal, a partir da raiz:

```sh
cd sudoku
mingw32-make
```

Se tiveres `make` disponível no teu ambiente:

```sh
cd sudoku
make
```

## Como executar

Dentro de `sudoku/`:

```sh
./sudoku map1.txt
./sudoku map2.txt
```
